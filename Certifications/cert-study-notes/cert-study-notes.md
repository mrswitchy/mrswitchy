# Cert Study Notes

> CompTIA Security+ and Network+ study materials

---

## 📋 Overview

This repository contains my personal study notes, domain breakdowns, cheat sheets, and practice Q&A for the **CompTIA Security+** and **CompTIA Network+** certifications — both of which I have earned. These notes are organized by exam domain and written to be clear, concise, and actually useful for review.

Feel free to use them for your own study.

## 🔐 Security+ (SY0-701) — Domain Breakdown

| Domain | Weight | Topics |
|---|---|---|
| 1. General Security Concepts | 12% | Controls, cryptography, authentication |
| 2. Threats, Vulnerabilities & Mitigations | 22% | Malware, social engineering, attacks |
| 3. Security Architecture | 18% | Cloud, network design, infrastructure |
| 4. Security Operations | 28% | IAM, monitoring, incident response |
| 5. Security Program Management & Oversight | 20% | GRC, policy, risk, compliance |

## 🌐 Network+ (N10-009) — Domain Breakdown

| Domain | Weight | Topics |
|---|---|---|
| 1. Networking Concepts | 23% | OSI model, protocols, ports |
| 2. Network Implementation | 20% | Routing, switching, wireless |
| 3. Network Operations | 17% | Monitoring, documentation, policies |
| 4. Network Security | 20% | Firewalls, VPN, IDS/IPS, AAA |
| 5. Network Troubleshooting | 20% | Methodology, tools, connectivity |
